Penerapan Route

Cari file di Route/url.php

<?php
use System\Route\RouteManager as route;

route::get('/', 'HomeController@index')->name('home');

route::post('/post/{id}', 'HomeController@store');

route::delete('/remove', 'HomeController@destroy')->name('rm');

route::group(['name' => 'p.'],function(){

    route::post('/data/user', 'HomeController@user')->name('user');
    route::post('/data/admin', 'HomeController@admin')->name('admin');
    
});

Penjelasan

Parameter pertama adalah url yg nantinya akan di akses pada web browser, dan yg kedua adalah controller tujuan. HomeController adalah class beserta file yg akan di gunakan. sedangkan index adalah method yg akan di proses.

macam method yg tersedia

Method Status
route::get Tersedia
route::post Tersedia
route::put Experimental
route::delete Tersedia
