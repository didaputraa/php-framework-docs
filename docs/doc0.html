Persiapan

Download / clone file pada github https://github.com/didaputraa/php-framework

Set up

Buka file Config/App.php

Nama aplikasi kita

protected static $app = 'Name';

Perlu diperhatikan

protected static $production = false;
#tidak akan menampilkan error jika mode production adalah true

protected static $maintenance= false;
#akan menampilkan maintenance mode pada semua halaman website
#berguna jika dalam masa perbaikan website

protected static $session   = [
    'name'      => 'Native-Nusantara',
    'lifetime'  => 3600,
    'httpOnly'  => true,    // jika menggunakan https ,harap ganti dengan false
    'secure'    => false,   // jika menggunakan https, harap ganti dengan true
    'domain'    => 'localhost'
];

protected static $guards    = [
    'model'     => \App\User::class,
];
/*
    Ini adalah class atau model yg nantinya akan digunakan pada Auth
*/

Menampilkan data Config

use Config/App;

echo App::getApp(); //App your name
